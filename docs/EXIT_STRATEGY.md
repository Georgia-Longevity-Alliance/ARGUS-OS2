# ARGUS-OS2 — ВЫХОД ИЗ ТУПИКА

**Date:** 2026-07-21
**Проблема:** 9 рецензий. SAI неспецифичен для СДА. Odf2-KO нарушает всё сразу.
**Решение:** Centriolin-KO вместо Odf2-KO.

---

## Почему Centriolin-KO решает все проблемы

| Проблема (с Odf2-KO) | Решение (Centriolin-KO) |
|-----------------------|--------------------------|
| SAI неспецифичен | Centriolin — ТОЛЬКО СДА. Odf2 на центросоме сохранён. |
| Cenexin-маркер потерян → нужен Dendra2 | **Cenexin (Odf2) на месте.** Возраст центросомы определяется штатно. |
| Δ4/5 не валидирован в RPE1 | **Не нужен.** Centriolin — эндогенный белок, KO специфичен. |
| Plk1-связывание vs СДА — не разделить | В Centriolin-KO Plk1-связывание (через Odf2) сохранено. Эффект → через СДА. |
| 9 рецензий, бесконечные правки | Простой дизайн: 3 группы. |

## Новый дизайн

```
ГИПОТЕЗА: Centriolin (СДА-специфичный белок) необходим для SAI.
Centriolin-KO → СДА теряются → SAI падает.
Centriolin-FL rescue → СДА восстанавливаются → SAI восстанавливается.
```

**Primary endpoint:** SAI. Измеряется с Cenexin (работает в Centriolin-KO! Odf2 на месте).

**Специфичный функциональный тест:** MT regrowth (нокодазол washout) — прямое измерение СДА-функции.

## Что Centriolin-KO доказывает

- SAI падает → СДА необходимы для асимметрии веретена ✅
- SAI восстанавливается с CNTRL-FL → эффект специфичен ✅
- SAI НЕ падает → СДА не нужны (→ другой механизм) ✅
