## 0. Суть

> **Центриоль — не просто «старая или новая». Центриоль — физический носитель записи всей centriole pedigree делений, через которые она прошла, включая пространственную ориентацию каждого шага.**

Каждое клеточное деление имеет направление: верх-низ, право-лево, перед-зад. Центриоль, проходя через дочернюю клетку, оказывается в конкретной пространственной позиции. Эта позиция оставляет структурный след. После N делений центриоль несёт запись: **«верх → право → зад → низ → ...»** — полную историю.

---

## 1. Почему это может работать

### 1.1. Механизм записи

Центриоль — полярная структура. У неё есть:
- **Дистальный конец** (appendages, цилия) — направлен к мембране
- **Проксимальный конец** (cartwheel) — направлен к ядру

При каждом делении центриоль занимает определённую позицию относительно осей клетки. Это положение влияет на:
- Ориентацию satellites (белки PCM, распределённые асимметрично)
- Паттерн полиглутамилирования тубулина (модификация, накапливающаяся со временем)
- Позицию appendages относительно предыдущей оси деления
- Ассоциированные РНК (центросомальная РНК)

**Каждая смена ориентации оставляет отметку.** Совокупность отметок = centriole pedigree.

### 1.2. Почему centriole pedigree предсказывает судьбу

В эмбриогенезе пространственная позиция клетки **определяет** её судьбу:
- Клетка наверху → эктодерма
- Клетка внизу → энтодерма
- Клетка слева → левый орган

Если центриоль записывает пространственную centriole pedigreeию, то прочитав центриоль мы знаем где была клетка → знаем кем она станет.

```
Центриоль: верх→право→зад → клетка была в дорсально-правой позиции → кожа
Центриоль: низ→лево→перед → клетка была в вентрально-левой позиции → мышца
```

## 2. Чем это отличается от известного

| Известное | Данная гипотеза |
|-----------|-----------------|
| Центриоль = возраст (старая/новая) | Центриоль = **полная centriole pedigree** (N шагов с ориентацией) |
| Anderson 2009: старая центриоль → быстрее цилия | Centriole pedigreeия A→B→C ≠ centriole pedigree X→Y→Z, даже если возраст одинаков |
| Thomas 2024: Cenexin → Plk1 → SAI | Пространственная ориентация satellites → предсказанная судьба |
| Дифференцировка = градиенты морфогенов | Дифференцировка = чтение центриолярной записи позиционной истории |

## 3. Что это даёт

### 3.1. Предсказание вместо возмущения

ARGUS-OS2 застрял в 9 рецензиях пытаясь доказать причинность через нокауты. Эта концепция обходит проблему: **нам не нужно нарушать.** Нам нужно читать.

Если корреляция «centriole pedigree центриоли ↔ судьба клетки» ≥95% — этого достаточно для предсказания. Причинность вторична.

### 3.2. Прогениторные карты

Зная centriole pedigreeию каждой центриоли, мы строим **ГРАФ** (не дерево):

```
Зигота (1 центриоль)
  ├─ верх → 2 центриоли
  │   ├─ право → кожа
  │   └─ лево → нейрон
  └─ низ → 2 центриоли
      ├─ зад → герминальная линия
      └─ перед → нейрон  ← КОНВЕРГЕНЦИЯ!
```

> **Одна судьба (нейрон) достигнута двумя разными centriole pedigreeми:** «верх→лево» и «низ→перед». Центриоли в этих двух клетках будут структурно идентичны — доказательство конвергенции.

**Прогениторная карта = граф с классами эквивалентности centriole pedigrees.**

Самообновление стволовой клетки: она ВСЕГДА получает центриоль с определённой centriole pedigree. Нарушение centriole pedigree → потеря стволовости.

### 3.3. Универсальность

Механизм не зависит от:
- Вида (C. elegans, Drosophila, мышь, человек)
- Ткани (нейроны, кожа, кровь)
- Молекулярных деталей (Odf2, Plk1 — это реализация, не принцип)

Принцип: **пространственная история → структурная запись → предсказанная судьба.**

## 4. Как ARGUS это проверяет

### 4.1. Эксперимент

| Шаг | Действие |
|:---:|----------|
| 1 | RPE1-hTERT Centrin1-GFP. 3D time-lapse. |
| 2 | Для каждого митоза: записать ориентацию веретена (XYZ-вектор) |
| 3 | Для каждой дочерней клетки: записать какую центриоль она получила |
| 4 | Построить дерево: каждый узел = клетка, каждое ребро = направление деления |
| 5 | Конечная точка: судьба клетки (цилия, маркер дифференцировки) |
| 6 | **Анализ:** коррелирует ли centriole pedigree центриоли с судьбой? |

### 4.2. Что измеряем

| Переменная | Метод |
|------------|-------|
| Centriole pedigreeия центриоли | Centrin1-GFP tracking + spindle vector (SiR-Tubulin) |
| Пространственная ориентация | 3D-координаты веретена относительно предыдущего деления |
| Структура центриоли | Super-resolution: satellites, polyglutamylation (GT335) |
| Судьба клетки | Цилия (Arl13B), дифференцировка (маркеры) |

### 4.3. Статистика

```
H₀: P(судьба А | centriole pedigree X) = P(судьба А | centriole pedigree Y)
H₁: Centriole pedigreeия предсказывает судьбу

Метод: условная энтропия. Сколько информации о судьбе содержится в centriole pedigree?
I(судьба ; centriole pedigree) = H(судьба) - H(судьба | centriole pedigree)
```

## 6. Если гипотеза верна

1. **Каждая центриоль — уникальный штрих-код.** Можно прочитать её историю.
2. **Судьба клетки предсказуема** по centriole pedigree центриоли с точностью >90%.
3. **Стволовость = определённая centriole pedigree.** Нарушение centriole pedigree → дифференцировка.
4. **Прогениторные карты** строятся автоматически: отследил центриоли → знаешь кто кем станет.
5. **Рак = сбой чтения centriole pedigree.** Клетка «забывает» свою позиционную историю.
