## Summary
ARGUS-OS2 is the **causality phase** built on the ARGUS-LP_OS v1.0 platform. Same hardware, new experiment: Odf2 knockout + HDAC6i rescue to test whether centrosome maturation state (Cenexin) is CAUSAL for daughter cell fate divergence. No laser. No micromanipulator. Observation only — but with genetic perturbation.
